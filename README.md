# אופטיקה פוקוס - דמו אתר לאופטיקאים

אתר תדמית/מכירות חד-עמודי (HTML/CSS/JS סטטי, RTL) שנבנה כדמו למכירה לחנויות אופטיקה.

## הקונספט
"להרכיב משקפיים" כחוויית גלישה:
- **הירו:** העולם מטושטש; עדשה עגולה עוקבת אחרי הסמן/אצבע וחושפת חדות. כפתור "הרכיבו משקפיים" מחדד הכל.
- **קולקציה:** ריחוף/נגיעה במסגרת מחדד אותה ומטשטש את השאר (אפקט פוקוס).
- **המרות:** קביעת תור לבדיקת ראייה (טופס ← וואטסאפ), וואטסאפ צף, ושאלון "התאמת מסגרת אונליין" (3 שאלות ← המלצה ← וואטסאפ).

## קבצים
- `index.html` - כל הסקשנים
- `css/styles.css` - מערכת עיצוב (שנהב/דיו/ענבר, Frank Ruhl Libre + Assistant)
- `js/main.js` - קטלוג מוצרים, אפקט עדשה, שאלון, טופס, מונים
- `IMAGE-PROMPTS.md` - 12 פרומפטים ל-Gemini + שמות קבצים ל-`assets/img/`
- `_serve.js` - שרת תצוגה מקומי (dev בלבד): `node _serve.js` ← http://localhost:4180

## התאמה ללקוח
- מספר וואטסאפ: `WA_NUMBER` בראש `js/main.js` (פורמט 9725XXXXXXXX)
- טלפון: חיפוש `+97235550000` ב-`index.html`
- מוצרים ומחירים: מערכי `EYEGLASSES` / `SUNGLASSES` ב-`js/main.js`
- כתובת ושעות: סקשן `#visit` ב-`index.html`

## פרסום (GitHub Pages)
1. להשלים את התמונות לפי `IMAGE-PROMPTS.md`
2. `git init`, commit, פוש לריפו חדש
3. להוסיף קובץ `.nojekyll` ריק
4. Settings → Pages → Deploy from branch → main / root

הערה: התמונות שטרם נוצרו מקבלות fallback אוטומטי (placeholder.svg לכרטיסים, גרדיאנט להירו).
